## 标题与段落：
在HTML中，每个段落都必须用`<p>`元素包裹：
```
<p>I am a paragraph, oh yes I am.</p>
```
每个标题都必须包裹在一个标题元素中:
```
<h1>I am the title of the story.</h1>
```
有六个航向元素：h1、h2、h3、h4、h5 和 h6。每个元素代表文档中不同层级的内容; 表示主标题，表示子标题，表示子标题，依此类推
## 结构层级的实现：
- 最好是每页使用一个标题——这是顶层标题，其他标题在层级结构中都位于此之下
- 确保你在层级中正确使用标题顺序。不要用元素表示子标题，然后再用元素表示子标题——那样不合理，还会导致奇怪的结果
- 在六个标题等级中，除非你觉得有必要，否则每页应尽量不超过三个。具有多层级的文档（例如深标题层级结构）会变得笨重且难以导航。在这种情况下，建议尽可能将内容分散到多个页面

## CSS样式
```
<h1>This is a top level heading</h1>
```
默认情况下，浏览器会给它设置大字号，使其看起来像标题,不过也可以用CSS把它样式成想要的任何样式

另一方面，你可以让任何元素看起来像顶级标题:
```
<span style="font-size: 32px; margin: 21px 0; display: block;">
  Is this a top level heading?
</span>
```
这是一个`<span>`元素。它没有语义上的区别。当想应用 CSS 给内容（或者用 JavaScript 做点什么）时，用它来包装内容，但不会赋予它额外的意义
