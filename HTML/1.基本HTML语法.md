## 什么是HTML：
HTML（超文本标记语言）是一种标记语言，用于告诉浏览器如何构建你访问的网页。HTML由一系列元素组成，你可以用它们包裹、包裹或标记内容的不同部分，使其呈现或以特定方式表现。包裹标签可以将内容变成链接，指向另一个页面，斜体表示词语等。

## HTML元素的结构：
```
<p>My cat is very grumpy</p>
```
- 开场标签`<p>`：这由**元素名称**（在本例中为段落的p）组成，并用开闭括号包裹。这个开头标签标记了该元素的起始或开始生效的时间。在这个例子中，它位于段落正文开始之前。
- 内容：这就是该元素的内容。在这个例子中，是段落文本——“我的猫很暴躁”。
- 结尾标签`</p>`：这与开头标签相同，只是元素名称前加了前斜杠。这标志着该元素的终点。未包含结尾标签是新手常见的错误，可能导致奇怪的结果。

### 嵌套元素：
元素可以放置在其他元素中。这称为嵌套。
```
<p>My cat is <strong>very</strong> grumpy.</p>
```
标签必须开合，使其彼此内或外侧。

### 空态元素:
并非所有元素都遵循开头标签、内容和结尾标签的模式。有些元素由单个标签组成，通常用于在文档中插入或嵌入某些内容。这类元素称为空元素，意为“不能包含其他HTML内容的元素”。

例如，`<br>`元素在文本行中插入换行，使其分隔到多行

>在一些HTML示例中，你会看到在空元素标签末尾添加一个/，比如`<br />`。这是一种不同的标记语法，虽然没错，但不需要这个“闭尾斜线”

### 属性：
```
<p class="editor-note">My cat is very grump</p>
```
属性包含了关于元素的额外信息，这些信息不属于其内容。类属性(class)提供了一个识别名称，可用于向元素提供样式（CSS）或脚本信息（JavaScript）。

**属性应具备：**

- 它和元素名称之间要有空格。当一个元素有多个属性时，属性之间也应该用空格分隔。
- 属性名称，后接一个等号=
- 一个属性**值**，用开引号和闭合引号包裹(即"")

### 向元素添加属性:
例如：`<img>`
其属性：
- src： 指定图像的URL（网址）的必要属性。例如：src="https://mdn.github.io/shared-assets/images/examples/fx-nightly-512.png"
- alt： 指定文本描述，向无法看到的人描述图像。例如： 。该属性技术上并非必需，但你确实应为所有传达意义的图像提供文本描述（而非纯粹装饰性）。alt="The Firefox Nightly icon"
- width： 表示图像的像素宽度。例如： 。width="300"
- height： 表示图像的高度（像素单位）。例如： 。height="300"

### 布尔属性：
有时你会看到没有值的 HTML 属性。这些被称为布尔属性。当添加布尔属性时，其值被设置为true ，无论赋予它什么值（甚至没有值）。如果某个属性未包含在HTML标签中，其值将设置为false 

例如，考虑 disabled 属性，你可以将其赋值给`<input>` 元素，阻止用户输入数据
```
<label for="first-input">This input is disabled</label>
<input id="first-input" type="text" disabled="disabled" />
<br />
```
>可以简写：`<input id="second-input" type="text" disabled />`


### 省略属性值周围的引号
在某些情况下，在属性值附近省略引号是可以的。但由于这可能会在其他情况下破坏加价，建议始终包含引号

- 没有空格时，可以省略下面显示的属性值周围的引号，且不会有负面影响
- 在带有空格的属性值中省略引号时，很快就会遇到问题

### 引号
单双引号都可以使用，只要保证使用过程中保持统一

要在同类型的其他引号（单引号或双引号）中使用引号，可以使用字符引用

`<a href="https://www.example.com" title="An "interesting" reference">A link to my example.</a>`

错误示例

`<a href="https://www.example.com" title="An &quot;interesting&quot; reference">A link to my example.</a> `

正确示例

## HTML文档的结构
```
<!doctype html>
<html lang="en-US">
  <head>
    <meta charset="utf-8" />
    <title>My test page</title>
  </head>
  <body>
    <p>This is my page</p>
  </body>
</html>
```
1. `<!doctype html>`：doctype文档声明头。在HTML刚兴起（1991-1992年）时，文档类型旨在作为链接，指向HTML页面必须遵守的一套规则，才能被视为优质HTML。文档类型以前大概是这样的：
```
<!doctype html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
```
在现代，文档类型是一个历史文物，必须包含它才能让其他所有功能正常工作。 是被视为有效文档类型的最短字符串，应包含在所有网页顶部。这就是你需要知道的全部
2. `<html></html>`： `<html>`元素。该元素将页面上的所有内容包裹起来。它有时被称为根元素。

3. `<head></head>`：<头>元素。该元素作为容器，存放用户将看到的内容之外的页面信息。这可以包括关键词和页面描述，用于搜索结果，CSS用于内容样式，字符集声明等。

4. `<meta charset="utf-8">`：一个<元元素>。该元素代表描述页面的元数据。字符集属性指定了文档将使用的字符编码。UTF-8包含绝大多数人类书写语言的字符，这意味着页面能够成功显示不同语言。没有理由不设置，这还能帮助避免后续问题。

5. `<title></title>`：<标题>元素。它会设定页面标题，也就是页面加载时浏览器标签页中出现的标题。当页面被收藏时，页面标题也用来描述该页面。

6. `<body></body>`：<身体>元素。它包含页面上显示的所有内容，包括文本、图片、视频、游戏、可播放的音轨等。

### HTML中的空白：
在几乎所有元素中（有例外，比如`<pre>`），无论你在HTML元素内容中使用多少空白，HTML解析器在渲染代码时都会将每个空白序列简化为单一空格。

### 字符引用：包括HTML中的特殊字符
这些是代表字符的特殊代码，用于特定情况下。每个字符引用以&号开头，以分号;结尾。
 | 字面特征 | 字符参考等价 |
 | :------: | :----------: |
 |   `<`    |    `&lt;`    |
 |   `>`    |    `&gt;`    |
 |   `"`    |   `&quot;`   |
 |   `'`    |   `&apos;`   |
 |   `&`    |   `&amp;`    |


###  HTML注释
HTML有一种机制可以在代码中写注释。浏览器会忽略评论，因此用户对评论是看不见的。注释的目的是让你在代码中加入注释，解释它是如何工作的。如果你离开代码库时间足够长以至于不记得它，或者有人开始写但从未见过，这非常有用。

要写HTML注释，请用特殊标记和，如下所示，将其包裹起来：`<!---->`

>快捷键：ctrl + /

```
<p>I'm not inside a comment</p>
<!-- <p>I am!</p> -->
```
