## 引言:
HTML包含可用于标记引文的功能;用哪个元素取决于在标记块还是内联引文

### 引用块:
如果某个段落层级内容（无论是一段、多段、列表等）是从其他地方引用的，你应该用`<blockquote>`元素包裹以表示，并在引用属性中包含指向引用来源的URL
```
<p>
  The <strong>HTML <code>&lt;blockquote&gt;</code> Element</strong> (or
  <em>HTML Block Quotation Element</em>) indicates that the enclosed text is an
  extended quotation.
</p>
                                        |
                                        |
                                        |
                                        |
                                        |
                                        |
                                       \/
<p>Here is a blockquote:</p>
<blockquote
  cite="https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/blockquote">
  <p>
    The <strong>HTML <code>&lt;blockquote&gt;</code> Element</strong> (or
    <em>HTML Block Quotation Element</em>) indicates that the enclosed text is
    an extended quotation.
  </p>
</blockquote>
```
>`<code>`标签在 HTML 中专门用于标记计算机代码或计算机相关的文本

>浏览器默认会将`<code>`包裹的内容渲染为等宽字体,并且通常字号会稍微小一点

浏览器默认样式会将此段(bockquote内部的)显示为**缩进段落**，作为引用的指示

### 内文引言：
内联引号的工作原理完全相同，只是它们使用了`<q>`元素
```
<p>
  The quote element — <code>&lt;q&gt;</code> — is
  <q
    cite="https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/q">
    intended for short quotations that don't require paragraph breaks.
  </q>
</p>
```
浏览器默认样式会将此表示为**普通文本**，并用引号表示引用
