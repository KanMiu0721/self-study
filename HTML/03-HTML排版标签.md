## 标题标签
- 标题使用`<h1>`至`<h6>`标签进行定义。`<h1>`定义最大的标题，`<h6>`定义最小的标题。具有align(用于对齐，但是HTML5中已经废除)属性，属性值可以是：left、center、right。
```
  <!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<meta name="viewport" content="width=device-width, initial-scale=1.0">
	<title>Document</title>
</head>
<body>
	<h1>H1：千古壹号，永不止步</h1>
	<h2>H3：千古壹号，永不止步</h2>
	<h3>H3：千古壹号，永不止步</h3>
	<h4>H4：千古壹号，永不止步</h4>
	<h5>H5：千古壹号，永不止步</h5>
	<h6>H6：千古壹号，永不止步</h6>
</body>
</html>

```
## HTML注释：
```
<!-- 我是 html 注释  -->
```
- 1. “行注释”快捷键Windows :  **Ctrl + /**

        使用技巧：
        - 单行注释：光标停留在这一行任何位置，直接按 Ctrl + /，它会自动在整行外面套上 ``。
        - 多行注释：鼠标选中多行，按下 Ctrl + /，这几行会同时被注释掉。
        - 取消注释：对已经注释的代码再次按下该组合键，注释就会消失。
  
- 2. “块注释”快捷键 (如果你只想注释某一行里的一部分单词，而不是整行)Windows: **Shift + Alt + A**
