# 文本格式化 
Ctrl + B：加粗 (Bold)
Ctrl + I：斜体 (Italic)
Alt + S：~~删除线~~ (Strikethrough)
Ctrl + Shift + ]：提高标题级别（比如把 ## 变成 ###）
Ctrl + Shift + [：降低标题级别
# 列表与链接操作
Enter 键：在列表项（- 或 1.）末尾按回车，会自动生成下一个列表符号。
Tab 键：在列表项开头按 Tab，会自动缩进（变成子列表）。
Shift + Tab：取消缩进。
Ctrl + L：快速插入链接 []( )。如果你先选中了一段文字再按，它会自动把文字放进方括号。
# 只有高手才知道的“骚操作”
自动补全表格：
在 Markdown 里画表格很痛苦。用了插件后，你只要写好表头（如 |标题1|标题2|）然后按回车，它会自动帮你对齐并补全下面的虚线。
数学公式 (LaTeX)：
虽然前端初期用不到，但你是 985 的学生，写实验报告或算法笔记肯定要用公式。
输入 $ 即可包裹行内公式，如 $E=mc^2$。
输入 $$ 即可包裹块级公式。
制作目录 (TOC)：
按下 Ctrl + Shift + P，输入 Markdown All in One: Create Table of Contents，它会自动根据你的标题生成一个带超链接的目录。
